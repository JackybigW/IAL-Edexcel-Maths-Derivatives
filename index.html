Part 1: Revision (Brief Overview)

1. Basic Derivative Formulas

Power Rule: If $f(x) = x^n$, then $f'(x) = nx^{n-1}$

Constant Rule: If $f(x) = c$, then $f'(x) = 0$

Sum Rule: $(f + g)' = f' + g'$

Product Rule: $(fg)' = f'g + fg'$

Chain Rule: $\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$

Quick Examples:

  • $f(x) = 3x^4 \Rightarrow f'(x) = 12x^3$
  • $f(x) = 2x^3 + 5x - 7 \Rightarrow f'(x) = 6x^2 + 5$
  • $f(x) = \sqrt{x} = x^{1/2} \Rightarrow f'(x) = \frac{1}{2}x^{-1/2} = \frac{1}{2\sqrt{x}}$

2. Gradients, Tangents and Normals

Gradient at point $(a, f(a))$:

$m = f'(a)$


Tangent line equation:

$y - f(a) = f'(a)(x - a)$


Normal line equation:

$y - f(a) = -\frac{1}{f'(a)}(x - a)$

3. Second Derivative

If $f'(x)$ is the first derivative, then:

$f''(x) = \frac{d}{dx}[f'(x)] = \frac{d^2y}{dx^2}$


Example: If $f(x) = x^4 - 2x^3 + x^2$

Then $f'(x) = 4x^3 - 6x^2 + 2x$

And $f''(x) = 12x^2 - 12x + 2$

Part 2: Today's Class - Advanced Applications

1. Increasing and Decreasing Functions

Key Principle: The sign of the first derivative determines if a function is increasing or decreasing.

For a function $f(x)$ on interval $(a,b)$:

  • If $f'(x) > 0$ for all $x \in (a,b)$, then $f$ is increasing on $(a,b)$
  • If $f'(x) < 0$ for all $x \in (a,b)$, then $f$ is decreasing on $(a,b)$
  • If $f'(x) = 0$, then $f$ has a stationary point

Step-by-Step Method:

  1. Find $f'(x)$
  2. Solve $f'(x) = 0$ to find critical points
  3. Test the sign of $f'(x)$ in each interval
  4. Determine where function increases/decreases

Example: Analyze $f(x) = x^3 - 3x^2 + 2$

Step 1: $f'(x) = 3x^2 - 6x = 3x(x - 2)$

Step 2: Critical points: $x = 0$ and $x = 2$

Step 3: Test intervals:

  • $x < 0$: $f'(-1) = 3(-1)(−3) = 9 > 0$ → Increasing
  • $0 < x < 2$: $f'(1) = 3(1)(-1) = -3 < 0$ → Decreasing
  • $x > 2$: $f'(3) = 3(3)(1) = 9 > 0$ → Increasing

2. Stationary Points and Their Nature

Stationary Points: Points where $f'(x) = 0$


Classification using Second Derivative Test:

  • If $f''(x) > 0$: Local Minimum
  • If $f''(x) < 0$: Local Maximum
  • If $f''(x) = 0$: Point of Inflection (or inconclusive)

Complete Method for Finding and Classifying Stationary Points:

  1. Find $f'(x)$ and solve $f'(x) = 0$
  2. Find $f''(x)$
  3. Evaluate $f''(x)$ at each critical point
  4. Classify each point using the second derivative test
  5. Find the y-coordinates: $y = f(x)$

Detailed Example: $f(x) = x^4 - 4x^3 + 4x^2$

Step 1: $f'(x) = 4x^3 - 12x^2 + 8x = 4x(x^2 - 3x + 2) = 4x(x-1)(x-2)$

Critical points: $x = 0, 1, 2$

Step 2: $f''(x) = 12x^2 - 24x + 8$

Step 3 & 4: Evaluate $f''(x)$ at critical points:

  • $f''(0) = 8 > 0$ → Local minimum at $(0, 0)$
  • $f''(1) = 12 - 24 + 8 = -4 < 0$ → Local maximum at $(1, 1)$
  • $f''(2) = 48 - 48 + 8 = 8 > 0$ → Local minimum at $(2, 0)$

3. Sketching Gradient Functions

Key Relationship: The gradient function $f'(x)$ shows the slope of the original function $f(x)$ at every point.

Important Connections:

  • Where $f(x)$ has maximum/minimum → $f'(x) = 0$
  • Where $f(x)$ is increasing → $f'(x) > 0$
  • Where $f(x)$ is decreasing → $f'(x) < 0$
  • Where $f(x)$ has steepest slope → $f'(x)$ has maximum/minimum

Original Function $f(x) = x^3 - 3x^2 + 2$

Gradient Function $f'(x) = 3x^2 - 6x$

Steps to Sketch $f'(x)$ from $f(x)$:

  1. Identify where $f(x)$ has horizontal tangents (stationary points)
  2. Mark these as x-intercepts of $f'(x)$
  3. Determine where $f(x)$ is increasing/decreasing
  4. Sketch $f'(x)$ positive/negative accordingly
  5. Consider the steepness of $f(x)$ to determine the magnitude of $f'(x)$

4. Real-Life Applications of Differentiation

🚗 Optimization Problems

Problem: A rectangular field is to be enclosed with 200m of fencing. What dimensions give maximum area?

Solution Process:

Let width = $x$, then length = $\frac{200-2x}{2} = 100-x$

Area: $A(x) = x(100-x) = 100x - x^2$

$A'(x) = 100 - 2x$

Setting $A'(x) = 0$: $100 - 2x = 0 \Rightarrow x = 50$

$A''(x) = -2 < 0$ → Maximum

Answer: 50m × 50m gives maximum area of 2500m²

📈 Economics: Marginal Cost and Revenue

If $C(x)$ is the cost function and $R(x)$ is the revenue function:

  • Marginal Cost: $MC = C'(x)$
  • Marginal Revenue: $MR = R'(x)$
  • Profit Maximization: $MR = MC$ or $R'(x) = C'(x)$

🏗️ Engineering: Related Rates

Problem: A balloon is being inflated. When radius = 5m, it's increasing at 2 m/s. How fast is volume changing?

Solution:

$V = \frac{4}{3}\pi r^3$

$\frac{dV}{dt} = \frac{dV}{dr} \cdot \frac{dr}{dt} = 4\pi r^2 \cdot \frac{dr}{dt}$

When $r = 5$ and $\frac{dr}{dt} = 2$:

$\frac{dV}{dt} = 4\pi(5^2)(2) = 200\pi$ m³/s
